<h2>🌿 About Backend</h2>

<p>
This backend is built using <b>Node.js + Express</b> to power <b>KhetMitra</b>.  
It handles APIs, authentication, and AI-based farming assistance.
</p>

<hr/>

<p align="center">

🤖 Groq AI &nbsp;&nbsp;|&nbsp;&nbsp; 🔐 JWT Auth
